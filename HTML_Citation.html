q tag

To write something in quotes we can use q tags

        Example: 
        <q>Time is Money.</q>

cite tag:

If we want to display statemet which is told by some person then we can write persons name inside the cite tag.
Also we can use to show conversation. 

        Example:
        Kunal: Time is money.
        Ram:  Yes, you are right.

        Here Kunal and Ram are written inside cite tag.
