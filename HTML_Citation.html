# Blockquote Element: [ <blockquote> ]

  • <blockquote> element is used to indicate that a section of text is a quotation from another source.
  • The <blockquote> element is typically used for longer quotations, and it is a block-level element, meaning it creates a block-level container for the quoted text.
  • By default, the <blockquote> element in HTML may have some margin to visually distinguish it as a block-level quotation and separate it from the surrounding content

# cite Attribute:

  • The cite attribute is used with specific HTML elements, such as <blockquote> and <figure>, to specify the source or reference for the content within those elements.
  • It is typically used to provide a URL that points to the source of the content. This URL can link to an external webpage, a document, or any resource that serves as the source of the quoted or referenced material.
  • The cite attribute is used for indicating the source of the content in a machine-readable way, making it useful for search engines and accessibility.

# <cite> Element:

  • The <cite> element is used to mark up the title of a creative work, such as the title of a book, movie, song, or the name of an author or source.
  • It is typically used within the content to provide proper semantic meaning to the title or author's name and is usually styled accordingly.
  • The <cite> element is used for indicating the title or name of a creative work or the name of an author, rather than specifying the source of a quotation.

# <bdo> element:

  • The <bdo> element in HTML stands for "Bi-Directional Override
  • This element is particularly useful when dealing with languages that are written from right to left (such as Arabic or Hebrew)
  • The element has a dir attribute that you can use to specify the desired text direction.
  • ltr: Specifies that the text should be displayed from left to right, overriding the default directionality if it's right-to-left.
  • rtl: Specifies that the text should be displayed from right to left, overriding the default directionality if it's left-to-right.
