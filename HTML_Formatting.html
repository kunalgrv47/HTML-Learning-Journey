06. HTML TEXT FORMATTING

  • Text formatting in HTML refers to the process of applying various visual styles and layout options to text content to improve its presentation and readability.
  • HTML provides several elements and attributes for text formatting, allowing you to control aspects such as font size, font family, text alignment, emphasis, and more.

# Some common HTML elements used for text formatting:

  • <b> - Bold text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text (highlight text with yellow color)
  • <strong> - Important text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text
