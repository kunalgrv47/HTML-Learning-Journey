HTML STYLE ATTRIBUTE

1. Style Attribute is used to add style to the HTML element, such as background color of page, text color, font, size and many more.
2. Style attributes has various property, such as: 

            1. background-color: for page background color
            2. color: for text color
            3. font-family: for fonts
            4. font-size: for font size
            5. text-align: for text allignment 

            And many more
