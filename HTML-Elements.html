02. HTML ELEMENTS:

  • HTML elements are the basic building blocks that structure and define the content on a webpage.
  • By combining various HTML elements and arranging them in a specific order, you can create complex web pages with text, images, links, forms, and interactive features.
  • An HTML element is defined by an opening tag, some content, and a closing tag.

# HTML ELEMENT STRUCTURE:

  • Below figure shows an example of HTML paragraph element, Which consists of Opening tag, Content and Closing tag.
  • The opening tag: The opening tag is the first part of an HTML element and is enclosed within angle brackets (<>). It consists of the element's name, which indicates what type of content or structure it represents. For example, a paragraph element's opening tag is <p>, and an image element's opening tag is <img>.
  • Closing Tag: The closing tag also uses angle brackets, but it includes a forward slash before the element name. It marks the end of the element and looks like </p> for a paragraph or </img> for an image. Not all HTML elements have closing tags, but most do.
  • Content: The content of an HTML element is the actual information or data that the element represents. It can include text, images, links, or other nested HTML elements, depending on the element's purpose. For example, the content of a paragraph element (<p>) is the text you want to display within that paragraph.

Examples of HTML Element:

  1. <h1>My first heading</h1>
  2. <p>My first paragraph.</p>
Opening Tag Content Closing Tag
<h1> My first heading </h1>
<p> My first paragraph </p>

NOTE: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an closing tag!

# NESTED HTML ELEMENT:

  • Nested HTML elements refer to the practice of placing one HTML element inside another.
  • This nesting allows web developers to organize and structure content on a webpage effectively.

Example of HTML Nested Element:
  • The <html> element is the root element and it defines the whole HTML document.
  • Then, inside the <html> element there is a <body> element.
  • The <body> element defines the document's body.
  • Then, inside the <body> element there are two other elements: <h1> and <p>
  • The <h1> element defines a heading and the <p> element defines a paragraph.
  • Note: Each elements has its own opening and closing tag.
