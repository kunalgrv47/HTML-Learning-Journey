PARAGRAPH TAG: [ < p > < /p > ]

1. In paragraph tag we can write the text.
2. < p > : Paragrapgh open tag.
3. < /p > : Paragraph closing tag.
4. In between opening and closing tag we can write the content and to break the line we can use "br tag". < br >

PRE TAG:

1. Prev tag is used to display the text exactly as we write in html file.
2. If we give enter in html then it will display enter in browser without giving "br" tag.
