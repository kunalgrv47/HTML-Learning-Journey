05. HTML PARAGRAPH:

  • The HTML <p> element defines a paragraph.
  • A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.
  • With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code.
  • The browser will automatically remove any extra spaces and lines when the page is displayed:

Q: How to write paragraph in HTML ?

  • If we want to write paragraph then we need to write our content between paragraph open and closing tag.
  • <p> : Paragrapgh open tag.
  • </p> : Paragraph closing tag.
  • Note: While writing content inside paragraph, if we are giving multiple spaces and new line then the browser will automatically remove any extra spaces and lines when the page is displayed.

Q: If we want to give a line break or if we want to display the exact content as it is written in html code, then how can we do this?

  • We can do this by using :
  • HTML Line Break: For new Line
  • <pre> tag: For displaying exactly as per the HTML code (retaining spaces and new lines)

# HTML Line Breaks:

  • The HTML <br> element defines a line break.
  • Use <br> if you want a line break (a new line) without starting a new paragraph.
  • Example: In the below image we can conclude that <br> creates new line.

# PRE TAG:

  • Prev tag is used to display the text exactly as we write in html file.
  • The text inside a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.
Note:
  • <pre> tag is very useful in writing poems.
  • In poems we have to write small sentences and we need to give spaces between two stanza. In this case if we will use br tags for line break and spaces then it will be very difficult. Hence we use pre tag for writing poem.
