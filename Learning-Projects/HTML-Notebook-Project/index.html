HTML BASICS

  • HTML Stands for Hyper Text Markup Language.
  • It is used for creating Web Pages.
  • HTML describes the structure of web page.
  • HTML consists of series of element.
  • HTML element tells the browser how to display the content.

01. HTML DOCUMENT STRUCTURE:

  • HTML document consists of hierarchical structure that is composed of elements and tags.
  • These elements and tags are used to organize & present content on a webpage.
  • Below figure shows the basic structure of HTML Document.
  • It consists of 4 parts:
    1. DOCTYPE (Document type declaration)
    2. html element
    3. head element
    4. body element
  • head and body elements are inside the html element. This is called as nested element (element inside element).

# Understand HTML structure with basic code snippet:
  1. Document Type Declaration (<!DOCTYPE>):
    • The document starts with a <!DOCTYPE> declaration that specifies the type and version of HTML being used. This declaration helps the web browser interpret and render the document correctly.
  2. HTML Element (<html>):
    • The <html> element is the root element of the HTML document. It wraps all other HTML elements on the page. It typically has two child elements: <head> and <body>.
  3. Head Section (<head>):
    • The <head> section contains metadata and other information about the web page. It does not display any visible content on the web page itself. Common elements within the <head> section include:
      • <meta>: Specifies character encoding, authorship, and other metadata
      • <meta charset="utf-8">: This is a meta element. The charset attribute of meta element specifies the character encoding for your document as UTF-8, which includes most characters from the vast majority of human written languages.
        With this setting, the page can now handle any textual content it might contain. There is no reason not to set this, and it can help avoid some problems later.
      • <title>: Sets the title of the web page, which appears in the browser's title bar or tab.
      • <link>: It is used to Link an external stylesheets (CSS) or other resources.
      • <script>: It is used to Link or embeds JavaScript code.
      • <style>: Contains internal CSS styles.
  4. Body Section (< body>):
    • The < body > section contains the visible content of the web page, including text, images, links, forms, videos, games, playable audio tracks, or whatever else.
    • This is where you create the structure and layout of your web page.
