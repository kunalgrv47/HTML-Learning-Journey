Usemap Attribute

* Here we are using usemap attribute with img tag.

* The usemap attribute creates a relationship between the img tag and the map tag.

* In map tag we are using and attribute name with value as image-map.(image map is an image with clickable areas)

* Inside map tag we need to declare the co-ordinates using coords attribute of the image. We can generate image co-ordinates online on this website

* Inside map tag we have to use one more attribute onclick to call function which is written under script (written using java script)


Below is an image with 2 specific area. You will get alert for the clicked area.

* In this example, I have used 3 tags: img tag, map tag and script tag
* In img tag, inserted image and used usemap attribute to link img tag with map tag.
* In map tag we have to mention image cordinate for specific area and also call functions accordingly and display the alert.
* In script tag functions are written to display alert.
